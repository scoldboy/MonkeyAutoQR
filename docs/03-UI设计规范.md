# UI 设计规范

## 设计原则

- **简洁**：去除一切非必要元素，信息层级清晰
- **直观**：按钮和操作的含义一目了然，无需学习
- **轻量**：组件不抢夺页面原有内容的注意力

---

## 配色方案

| 角色 | 色值 | 用途 |
|------|------|------|
| 主色（Primary） | `#4A90D9` | 主按钮背景、链接文字、标题栏 |
| 主色浅（Primary Light） | `#EBF3FC` | 弹框标题栏背景、悬浮按钮背景 |
| 主色深（Primary Dark） | `#357ABD` | 主按钮 hover 态 |
| 成功色 | `#27AE60` | 复制成功提示 |
| 警告色 | `#F39C12` | 识别失败提示 |
| 文字色（Primary Text） | `#2C3E50` | 正文、标题 |
| 次要文字 | `#7F8C8D` | 辅助说明、链接 URL 展示 |
| 边框色 | `#DCDFE6` | 输入框边框、分隔线 |
| 白色 | `#FFFFFF` | 弹框内容区背景 |
| 遮罩色 | `rgba(0,0,0,0.4)` | 弹框遮罩层 |

## 组件规范

### 1. 扫描按钮（选区触发）

```
┌─────────────────────┐
│  🔍  扫描二维码      │
└─────────────────────┘
```

| 属性 | 值 |
|------|-----|
| 宽度 | `130px` |
| 高度 | `36px` |
| 圆角 | `8px` |
| 背景色 | `#EBF3FC` |
| 边框 | `1px solid #4A90D9` |
| 文字颜色 | `#4A90D9` |
| 字体大小 | `13px` |
| 字重 | `500` |
| 阴影 | `0 2px 8px rgba(74,144,217,0.2)` |
| hover 背景 | `#4A90D9` |
| hover 文字 | `#FFFFFF` |
| 定位 | `position: fixed`（相对于视口） |
| z-index | `99999` |
| 间距 | 距图片右上角外侧 8px |
| 触发方式 | 用户拖动选区覆盖图片后出现 |
| 消失方式 | 点击页面空白处 / 点击其他图片的按钮时消失 |

### 2. 结果弹框

```
┌─────────────────────────────────┐ ← 遮罩层 rgba(0,0,0,0.4)
│  ┌───────────────────────────┐  │
│  │  🔗  二维码扫描结果    [X] │  │ ← 标题栏 #EBF3FC
│  ├───────────────────────────┤  │
│  │                            │  │
│  │  https://example.com/...   │  │ ← 链接展示区
│  │                            │  │
│  ├───────────────────────────┤  │
│  │  [复制链接]   [打开链接]   │  │ ← 按钮区
│  └───────────────────────────┘  │
└─────────────────────────────────┘
```

| 属性 | 值 |
|------|-----|
| 弹框宽度 | `420px`（最大宽度，小屏自适应） |
| 圆角 | `12px` |
| 背景 | `#FFFFFF` |
| 阴影 | `0 8px 32px rgba(0,0,0,0.18)` |
| 标题栏高度 | `48px` |
| 标题栏背景 | `#EBF3FC` |
| 标题文字 | `14px`, `#2C3E50`, 居中 |
| 关闭按钮 | `32×32px`, 右侧, 文字色 `#7F8C8D` |
| 链接展示区 | 内边距 `20px` |
| 链接文字 | `14px`, `#4A90D9`, 自动换行, 可选中 |
| 按钮区 | 内边距 `0 20px 20px`, flex 水平排列, 间距 `12px` |
| z-index | `100000` |

### 5. 提示弹框（识别失败 / 跨域限制）

与结果弹框结构相同，但：
- 标题栏文字为「提示」
- 内容区显示提示文字而非链接
- 只有一个「确定」按钮（居中，宽度 `120px`）

## 动画

| 动画 | 时长 | 效果 |
|------|------|------|
| 按钮出现（选区触发） | 150ms | `opacity 0→1` + `transform scale(0.9→1)` |
| 按钮消失（点击空白处） | 150ms | `opacity 1→0` + `transform scale(1→0.9)` |
| 弹框出现 | 200ms | `opacity 0→1` + `transform scale(0.95→1)` |
| 遮罩出现 | 200ms | `opacity 0→1` |

## 字体

- 优先使用系统默认中文字体栈：`-apple-system, BlinkMacSystemFont, "Segoe UI", "PingFang SC", "Microsoft YaHei", "Helvetica Neue", sans-serif`
- 等宽字体（链接 URL 展示）：`"SF Mono", "Menlo", "Consolas", monospace`
