# MonkeyAutoQR

一款浏览器油猴脚本，拖动鼠标选中网页图片区域后出现扫描按钮，点击即可识别图片中的二维码，弹框展示链接。

## 功能

- 🖱️ **选区触发** — 像选文字一样拖动鼠标，覆盖图片即显示扫描按钮
- 📷 **二维码识别** — 基于 [jsQR](https://github.com/cozmo/jsQR)，纯前端识别
- 🔗 **一键复制 / 打开** — 扫码结果弹框，支持复制链接或新标签页打开
- 📐 **智能定位** — 按钮自动适配视口边界，不溢出屏幕
- 🌐 **跨域兼容** — 跨域图片自动通过 GM_xmlhttpRequest 降级获取

## 安装

1. 安装 [Tampermonkey](https://www.tampermonkey.net/) 或 [Violentmonkey](https://violentmonkey.github.io/) 浏览器扩展
2. 打开 [qr-scanner.user.js](qr-scanner.user.js)，点击 Raw 按钮，Tampermonkey 会自动弹出安装页面
3. 点击「安装」即可

## 使用

1. 浏览网页时，**拖动鼠标选中包含二维码图片的区域**
2. 被覆盖的图片右上角会出现「扫描二维码」按钮
3. 点击按钮即可识别，弹框展示链接
4. 点击空白处可隐藏按钮

## 注意事项

- 图片需 ≥ 64×64 像素，太小（如图标）不会触发
- SVG 格式图片不支持扫描
- 部分跨域图片可能因安全策略无法读取
